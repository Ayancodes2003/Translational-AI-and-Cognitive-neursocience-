Model Comparison for Depression Detection

Translational AI and Cognitive Neuroscience Project

Date: April 11, 2025

Project Overview

Objectives

  • Develop AI models for depression detection using multimodal data
  • Compare different deep learning architectures
  • Identify the most effective model for clinical applications

Approach

  • Implemented three different neural network architectures
  • Trained models on synthetic data (for demonstration)
  • Evaluated performance using standard metrics
  • Analyzed training behavior and computational efficiency

Model Architectures

SimpleNN

  • Simple feedforward neural network
  • One hidden layer with 64 neurons
  • ReLU activation and dropout
  • Computationally efficient

LSTM

  • Long Short-Term Memory network
  • 2 LSTM layers with 64 hidden units
  • Designed for sequential data
  • More computationally intensive

CNN

  • Convolutional Neural Network
  • 2 convolutional layers (16 and 32 filters)
  • Max pooling and dropout regularization
  • Fully connected layers for classification
  • Effective at capturing spatial patterns

Key Findings

1. CNN Superiority

The CNN model outperformed other architectures across all metrics, achieving 90% accuracy and 0.97 AUC.

2. Efficiency Considerations

SimpleNN offers a good balance between performance (85.5% accuracy) and computational efficiency (0.29s training time).

3. LSTM Limitations

The LSTM model performed poorly despite longer training time, suggesting sequential modeling may not be optimal for this task.

4. Clinical Implications

High recall (92.9%) of the CNN model is particularly important for clinical applications to minimize missed cases of depression.

Conclusion and Next Steps

Conclusion

The CNN architecture demonstrates superior performance for depression detection, likely due to its ability to capture spatial patterns in the data.
